# AI in Postgraduate Pulmonology Education Dataset

This repository contains the supporting dataset associated with the manuscript

> Evaluating Artificial Intelligence Integration in Postgraduate Pulmonology Education: A Multicenter Cross-Sectional Study

## Repository Structure

data/

- 01_participants.csv
- 02_content_validation.csv
- 03_reliability_cronbach_alpha.csv
- 04_perceived_educational_benefits.csv
- 05_ethical_concerns.csv
- 06_curricular_acceptance.csv
- 07_synthetic_questionnaire_responses.csv

## Description

The repository contains anonymized data used for the statistical analyses presented in the manuscript.

The questionnaire evaluates six educational dimensions:

- AI Literacy
- Educational Utility
- Pedagogical Impact
- Ethical Risks
- Curricular Acceptance
- General Perception
